Restage and Reload

If changes have been made to the data source configuration since the last load or the source data's schema has changed, then choose "Restage and reload".

Reload Only

Choosing "Reload only" will use previously staged data as long as it is available.

Reuse Schema

Choosing "Reuse schema" will reuse previously generated schemas for selected datasets. Note that reloading without schema regeneration may lead to missing columns or tables.
